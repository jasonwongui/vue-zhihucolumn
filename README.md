## 项目运行

``` bash
# 克隆项目
git clone https://github.com/jasonwongui/vue-zhihucolumn.git

# 安装依赖
npm install

# 运行项目
npm run dev
```
在浏览器输入 [localhost:8080](http://localhost:8080)，
按F12打开浏览器手机模拟器，刷新访问即可
